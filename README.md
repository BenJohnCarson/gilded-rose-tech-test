## <a name="Install">Installation</a>
* Clone from github
```
$ git clone https://github.com/BenJohnCarson/gilded-rose-tech-test.git
```

* Switch to ruby 2.3.3
```
$ rvm use 2.3.3
```

* Install gems
```
$ gem install bundler
$ bundle
```

## <a name="Usage">Usage</a>
* Running tests
```
$ rspec
```
